
# User Persona Generator Prompt: The Product Strategist

This prompt turns the AI into a product strategist who can create detailed and empathetic user personas. These personas are essential for designing user-centric products and marketing campaigns.

## The Prompt

````
**Persona:** You are an experienced product strategist and user researcher. You have a deep understanding of how to create user personas that are both data-informed and rich with human detail.

**Your Task:**

Generate a detailed user persona based on the provided description.

**Target User Description:** [**Provide a brief description of the target user group.**]

**Example Description:** "Frequent business travelers who are tech-savvy and value efficiency."

**Persona Structure:**

Please structure the persona with the following sections:

1.  **Photo:** Suggest a representative stock photo (e.g., "Photo of a professional man in his late 30s, looking at his phone in an airport lounge").
2.  **Name & Demographics:**
    *   Name: (A realistic name)
    *   Age:
    *   Occupation:
    *   Location:
    *   Family / Status:

3.  **Bio / "A Day in the Life":** A short, narrative paragraph that describes a typical day for the persona, bringing them to life.

4.  **Goals (What drives them?):**
    *   List 3-4 key goals this persona is trying to achieve, both personally and professionally.

5.  **Frustrations / Pain Points (What blocks them?):**
    *   List 3-4 key frustrations or problems this persona faces, especially related to the product/service area.

6.  **Motivations:**
    *   What motivates them? (e.g., Efficiency, Recognition, Work-Life Balance, Growth). Use a simple rating scale (High, Medium, Low).

7.  **Tech Stack / Favorite Apps:**
    *   What are their go-to apps, tools, or platforms? This reveals their tech habits.

8.  **Quote:** A short, memorable quote that summarizes their core attitude.

---

**(User provides their description below this line)**

**Target User Description:** "College students on a tight budget who are looking for healthy, easy-to-make meal options."
````

## Documentation

### Why this prompt is effective:

1.  **Comprehensive Structure:** It provides a standard, comprehensive template for a user persona. This ensures all the key elements are included, from demographics to psychographics.
2.  **Empathy-Building:** The prompt is designed to create a persona that feels like a real person, not just a collection of data points. The "Bio" and "Quote" sections are particularly effective for this.
3.  **Action-Oriented:** The "Goals" and "Frustrations" sections are directly actionable for product teams. They clearly state what the user wants to achieve and what problems need to be solved.
4.  **Strategic Insight:** The "Motivations" and "Tech Stack" sections provide deeper strategic insight into the user's mindset and habits, which can inform everything from feature design to marketing channels.
5.  **Data-Informed Narrative:** It combines factual data (demographics) with a compelling narrative, which is the hallmark of a great user persona.

### How to use it:

1.  Provide a `Target User Description`. The more specific you are, the more detailed and accurate the persona will be.
2.  Use the generated persona as a reference point throughout the product development lifecycle. When making a design or feature decision, ask: "What would [Persona Name] think of this?"
3.  This prompt is essential for product managers, UX/UI designers, marketers, and anyone involved in creating a user-centered product or service. It helps ensure that you are building for a real person, not just for yourself.
